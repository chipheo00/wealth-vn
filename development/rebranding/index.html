Wealthfolio → WealthVN Rebranding Plan

Date: November 30, 2025
Status: Complete

Overview​

This document tracks the rebranding of "Wealthfolio" to "WealthVN" for the Vietnamese market.

New Brand Identity​

Execution Phases​

Phase 1: Core Identity (CRITICAL)​

  • Update Rust Cargo.toml files (src-core, src-tauri, src-server)
  • Update root package.json and workspace packages
  • Update tauri.conf.json (productName, identifier, disable updater)
  • Update iOS/macOS project.yml and rename directories

Phase 2: Import Paths​

  • Global replace @wealthfolio/_ imports to @wealthvn/_
  • Update tsconfig.json and vite.config.ts paths

Phase 3: UI & Localization​

  • Update localization files (en/.json, vi/.json)

Phase 4: Docker & Server​

  • Update Dockerfile and environment files

Phase 5: External References​

  • Update/disable external URLs (addon store, website links)
  • Update GitHub repo URLs and author info

Phase 6: Documentation​

  • Update README.md and documentation files

Phase 7: Verification​

  • Verify build compiles (pnpm build, cargo check)

Notes​

  • Logo and icon assets should be replaced separately with new WealthVN branding
  • Database migration: existing users' database files will continue to work
  • The app will look for the same database path, just with different app identity

Post-Rebranding Checklist​

  • Replace logo files (public/logo.svg, public/logo.png)
  • Regenerate app icons for all platforms (src-tauri/icons/)
  • Update iOS assets (src-tauri/gen/apple/Assets.xcassets/)
  • Run pnpm install to update lockfile
  • Run pnpm build to verify frontend builds
  • Run cargo check to verify Rust compiles
  • Test pnpm tauri dev for desktop app
  • Commit and push to new repository
